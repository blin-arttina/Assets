# AssetsAlertNFT Deployment Guide

## Deployment Instructions (Remix)

1. Open [Remix](https://remix.ethereum.org).
2. Create a new file and paste the content from `AssetsAlertNFT.sol`.
3. Compile the contract using Solidity 0.8.x.
4. Deploy on Polygon using "Injected Provider - MetaMask".
5. After deploying, verify on Polygonscan using:
   - Compiler: 0.8.x
   - License: MIT
   - Optimization: Yes
